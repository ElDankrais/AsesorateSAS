# AsesorateSAS
Proyecto de ingeniería de requisitos, de un software para asesorías académicas

## Arquitectura

**Vistas:** 
 - HTML
 - CSS
 - Javascript
 
**Controlador**
 - Python 3 Django
 
**Codificación**
 - PEP8 Estandar
 
**Base de datos**
  - MySQL
  - **Usuario**: admin-admin
  - **Nombre base de datos**:asesorate_sas
